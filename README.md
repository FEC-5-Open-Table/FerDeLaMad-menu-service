# Project Name

> Menu component for restaurant reservation website

## API Documentation

* GET: '/api/restaurant/:id'
* POST: '/create/restaurant'
* PUT: '/update/restaurant'
* DELETE: '/delete/restaurant/:id'

## Requirements
* MongoDB v4.0.0
* Node v8.11.1
* npm 6.1.0

## Development Setup
1. Navigate to root directory
2. Run npm install
  `code(npm install)`
3. Seed the database
  `code(npm run seed)`
